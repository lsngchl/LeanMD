# Choosing a LeanMD Node Template

Use this guide only to choose the template that matches the primary question answered by the new node.

| Template | Use it when the node should answer |
|---|---|
| [Root node](./root.md) | What is the main theorem, and how do its principal ingredients combine? |
| [Result node](./nodes/result_node.md) | What precise theorem, proposition, lemma, or mapping statement holds, and why? |
| [Calculation node](./nodes/calculation_node.md) | Why does this particular identity, rank, kernel, derivative, or block formula hold? |
| [Concept node](./nodes/concept_node.md) | What does this definition or construction mean? |
| [Reference and application node](./nodes/reference_application_node.md) | What external or standard result is being used, and how does it apply here? |

If more than one template appears applicable, choose the one matching the main reason a reader would open the node.
